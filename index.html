¿Qué es CSS? 🎨

CSS (Cascading Style Sheets) es un lenguaje que te permite dar estilo a tus documentos HTML. A través de CSS, puedes controlar la apariencia de texto, colores, márgenes y más.

Ejemplo de Enlace a una Hoja de CSS 🔗

Puedes enlazar una hoja de estilos CSS utilizando el siguiente código en el encabezado de tu HTML:

<link rel="stylesheet" href="styles.css">
